# Basic Git Commands

| Command        | Description                          |
|----------------|--------------------------------------|
| `git status`   | Shows the status of your repo        |
| `git add .`    | Adds all changes to the staging area |
| `git commit -m "msg"` | Commits with a message         |
| `git push`     | Pushes changes to remote repository  |
| `git mv`       | Renames or moves a file              |
